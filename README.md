## About me :smiley:
My name is Luke. I'm a SWE that likes to build projects for fun using Claude code.

## Current Project
I'm currently working on a college football analytics project.

My goal is to create a mobile app that brings users more joy on college football Saturdays. At the end of the year, only one team can win the national title, but teams you **hate** lose every week! :smirk:

My app will include input from a model that I'm building to predict games against the spread (ATS), because some people are especially interested in that sort of thing. I have other feature ideas, but want to keep an open mind to let this process unfold naturally.
